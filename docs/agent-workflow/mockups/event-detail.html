This screen isolates the drawer pattern so it can be reviewed as a visual system component, not just as a sidebar.

Design intent

Trust block gets its own section

Verification, confidence, sources, and freshness are elevated into a dedicated section so users can judge credibility without hunting through generic metadata.

How this pattern is used

On desktop, this lives as a persistent right-side panel in Map, Timeline, and Priority Feed. On smaller screens, it can translate into a bottom sheet without changing section order.

Interaction notes

Selection should persist during background refresh where possible. New data should update freshness subtly instead of collapsing the user’s inspection context.

Fallback behavior

If detailed source metadata is unavailable, show an explicit note rather than implying certainty. Keep action slots visible with disabled or deferred states.
